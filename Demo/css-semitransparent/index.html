CSS半透明背景实现方案

#

实现（半）透明背景效果的代码

实现原理：低版本的IE浏览器使用渐变滤镜Gradient Filter，支持CSS3的浏览器使用rgba属性。

使用方法：输入 十六进制颜色值 及 透明度值 ，点击 生成代码 。其会自动在IE的过渡滤镜以及CSS3中的rgba属性之间进行十六进制与十进制的换算。

说明：

  1. 样式名

    例如： .head-bg ，填样式名选项后会自动导出IE9的兼容方案，不填样式名直接输出样式属性，请自己写IE9的hack。单独把IE9的 filter 变成透明度为0即可。高级浏览器大部分支持 :root 伪类，但不支持filter属性，而IE只有IE9支持，所以我们可以这样写。代码如下：

    :root .rgba{filter:progid:DXImageTransform.Microsoft.Gradient(GradientType=0, StartColorStr='#00000000', EndColorStr='#00000000');}
  2. 十六进制颜色值

    十六进制是从【0】到【9】以及【a】到【f】组合而成的，请检查你的颜色值是否为【三位】或者符合【十六进制的组合方式】。

  3. 透明度

    透明度的值在【0】到【1】之间。如： 0.5 。
